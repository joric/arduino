## Hardware

* MB102 Breadboard
* Pro Micro
* Cherry MX Blue
* Cherry doubleshot ESC
* TM1637 Segment Display Module
* EHC-SR04 ultrasonic sensor
* SPI SD Card shield
* Speaker Shield
* Li-Po batteries
* SG90 servo (1.8 kg/cm)
* RDS3115 servo (17 kg/cm)
